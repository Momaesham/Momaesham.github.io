## Что такое React

React — это библиотека JavaScript для создания пользовательских интерфейсов.

React нужен, когда:

* интерфейс часто меняется
* есть данные, которые влияют на отображение
* приложение состоит из повторяющихся частей

Пример:

* список игр
* каждая игра — одинаковый блок, но с разными данными

React решает это с помощью **компонентов**.

---

## Что такое компонент

Компонент — это:

* отдельная часть интерфейса
* функция, которая возвращает HTML (JSX)
* независимый и переиспользуемый блок

Примеры компонентов в нашем приложении:

* `App` — главный компонент
* `Header` — шапка приложения
* `GameList` — список игр
* `GameItem` — одна игра

Каждый компонент отвечает **только за свою задачу**.

---

## Архитектура приложения

React-приложение строится как дерево компонентов:

```
App
 ├─ Header
 └─ GameList
     ├─ GameItem
     ├─ GameItem
     └─ GameItem
```

`App` — корень приложения. Все остальные компоненты подключаются к нему.

---

## Что такое JSX

JSX — это синтаксис, который позволяет писать HTML прямо в JavaScript.

Пример:

```js
function Header() {
  return (
    <h1>Game Library</h1>
  )
}
```

Под капотом это обычный JavaScript.

## Подготовка окружения

Перед созданием проекта необходимо:

## Главный компонент App

`App` — это основной компонент, который:

* объединяет всё приложение
* содержит другие компоненты

Пример:

```js
function App() {
  return (
    <div>
      <h1>Game Library</h1>
    </div>
  )
}

export default App
```

---

## Создание компонентов

Создаём папку:

```
src/components
```

В ней будем хранить все компоненты приложения.

---

### Компонент Header

Файл: `Header.jsx`

Задача компонента:

* отображать название приложения

Пример:

```js
function Header() {
  return (
    <header>
      <h1>Game Library</h1>
    </header>
  )
}

export default Header
```

---

### Компонент GameItem

Файл: `GameItem.jsx`

Задача:

* отображать одну игру

Пока используем статический текст.

Пример:

```js
function GameItem() {
  return (
    <div>
      <h3>Cyberpunk 2077</h3>
      <p>Статус: В процессе</p>
    </div>
  )
}

export default GameItem
```

---

### Компонент GameList

Файл: `GameList.jsx`

Задача:

* отображать список игр
* содержать несколько `GameItem`

Пример:

```js
import GameItem from './GameItem'

function GameList() {
  return (
    <div>
      <GameItem />
      <GameItem />
      <GameItem />
    </div>
  )
}

export default GameList
```

---

## Сборка приложения

Обновляем `App.jsx`:

```js
import Header from './components/Header'
import GameList from './components/GameList'

function App() {
  return (
    <div>
      <Header />
      <GameList />
    </div>
  )
}

export default App
```

Теперь приложение состоит из компонентов.
